# atnow

Store and view you own murmur.

# Dependency

* Ruby
* Yad
* Zenity
* `xdg-user-dir`

# Usage

1. Start `atnow.rb`
2. Icon will be in systray.
3. Click icon.
4. Murmuring.
5. Right click icon.
6. Click long murmur.

# Advanced

* You can open murmuring window with HUPping this process.
* You can find this process's pid in `$home/.atnow.pid` file.
